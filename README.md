# BindSQL Java Logs

JavaアプリケーションのSQLログに出力されたSQLとバインド変数を、PostgreSQLクライアントで実行しやすいSQLへ変換するローカルツールです。

`BindSQLJavaLogs.html` の単一HTMLだけで動作します。外部CDNや外部APIには依存しません。

## 使い方

1. `BindSQLJavaLogs.html` をブラウザで開きます。
2. 左側の入力欄に、SQLログまたはSQLとバインド変数を貼り付けます。
3. 右側にPostgreSQL向けSQLがリアルタイムで表示されます。
4. `コピー` ボタンで結果をコピーできます。

## できること

- SQL本文とバインド変数をまとめて貼り付けて、PostgreSQLで実行しやすいSQLへ変換
- `?` の位置バインドと `:name` の名前付きバインドを実値へ置換
- SQLキーワードを大文字化
- 主要な句ごとに改行して整形
- インデントは半角スペース2つ単位
- ライト/ダークモード切り替え

## 整形方針

- SQL文字列、コメント、引用識別子、PostgreSQLのドルクォート文字列内のキーワードは整形対象から保護します。
- 未置換のバインド変数や余剰のバインド変数がある場合は、変換結果と一緒に警告を表示します。
- 出力SQLはPostgreSQLでの確認・実行を主対象としています。
- DB種類セレクトは今後の拡張用です。現状はPostgreSQLのみ選択できます。
- このツールはSQLログを実行しやすい形に整える補助ツールです。他DBで実行する場合は、対象DBの方言や環境に合わせて確認してください。
